プログラミング演習環境の準備¶

著者:倉光君郎
出版社:Packt Publishing

内容¶

本講義用に準備した演習システム Konoha をインストールします。 次の3ステップで実行します。

  1. Java 8 (最新版のインストール)
  2. Konoha のダウンロード
  3. Konoha の実行

Java 8 のインストール¶

Oracle のページから配布されているJava8 (JDK) をダウンロードして インストールします。

  • Oracle Java8 Download - http://www.oracle.com/technetwork/java/javase/downloads/index.html
  • Mac OS X 参考情報
  • Windows 補足情報

Konoha のインストール¶

Konoha は、jar ファイルとして配布されています。

  • Konoha のダウンロード - http://kkuramitsu.github.io/prog/konoha.jar
..note::
Konoha は頻繁に更新されます。毎回、新しいバージョンをダウンロードして使ってください。

Konoha の起動¶

ターミナル（コマンドプロンプト）から先ほどダウンロードをした konoha.jar を指定して起動する。

$ java -jar konoha.jar
