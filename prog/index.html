プログラミング入門 (2015年度版)

本年度は、プログラミングのサポートページです。 横浜国立大学で提供しているプログラミング入門のサポートページです。

授業計画

2015年度は、プログラミング言語に依存しない概念やアイディアを学んだあと 演習する構成で授業を進めます。

  • Day 1 - Variable, Operator, Function
  • Day 2 - Type, Function
  • Day 3 - Control Flow (If, While, Recursion)
  • Day 4 - Array (Dict)
  • Day 5 - Class, Object
  • Day 6 - Type, Function (Again)
  • Day 7 – Programming Language Design
  • Day 8 - C言語
  • Day 9 – 12 演習室
  • Day 13 Evaluation

プログラミング演習環境の準備

本講義用に準備した演習システム Konoha をインストールします。 次の3ステップで実行します。

  • ステップ1. Java 8 (最新版のインストール)
  • ステップ2. Konoha のダウンロード
  • ステップ3. Konoha の実行

Java 8 のインストール

Oracle のページから配布されているJava8 (JDK) をダウンロードして インストールします。
  • Oracle Java8 Download - http://www.oracle.com/technetwork/java/javase/downloads/index.html
  • Mac OS X 参考情報
  • Windows 補足情報

Konoha のインストール

Konoha は、jar ファイルとして配布されています。
  • Konoha のダウンロード - http://kkuramitsu.github.io/prog/konoha.jar
注意：Konoha は頻繁に更新されます。毎回、新しいバージョンをダウンロードして使ってください。

Konoha の起動

ターミナル（コマンドプロンプト）から先ほどダウンロードをした konoha.jar を指定して起動する。 
  $ java -jar konoha.jar
